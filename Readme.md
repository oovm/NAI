Novel AI 傻瓜版
=====================

只要爽就行了吧~~~

分享你的提示词, 让更多人来涩涩吧!

### 下载

打开 release 下载可执行文件.

在同目录下新建 nai.toml, 里面放你的 key.

不需要翻墙.

### 自动出图

双击或者命令行输入

```sh
nai.exe new  # 常规版
nai.exe ss   # 涩涩版
```

### 高级模式

```sh
nai.exe new [名称] [提示词] -t [线程数量] -n [图片数]
nai.exe new miku 'miku' -n -5 # 出五张 miku 的图
```
